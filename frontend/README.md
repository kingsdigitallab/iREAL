# iREAL - Frontend

The frontend is a dashboard to explore data extracted by the AI/ML processes
implemented in the backend.

The frontend is built with [SvelteKit](https://kit.svelte.dev).

## Getting started

1. Clone this repository
1. Install dependencies

    ```bash
    npm install
    ```

1. Run the development server

    ```bash
    npm run dev
    ```

1. The dashboard should be available at http://localhost:5173/.

### Building

To create a production version of the project:

```bash
npm run build
```

The production version can be previewed with `npm run preview`.
